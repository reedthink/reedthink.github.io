=_= 一个题如果一直超时，要考虑是不是有很多重复的计算。
POJ-2992

题意

计算组合数的因子个数。(0 ≤ k ≤ n ≤ 431)

分析

显然，我们不能算出组合数再求因子个数。求N的因子个数，可以将N分解质因数，每个质因子的数量A_i先+1，然后将所有A_i相乘，就是因子个数。别问，问就是不会证+_+
这个题目的查询蛮多的，打表不打表，差距不大。关键上述做法的实现是否够快
